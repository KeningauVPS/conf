Automatic Script Installer by Perantau Sepi (Izam Lukman)

==========

## Usage
### Centos 7 (OpenVZ VPS)
```
wget https://raw.githubusercontent.com/lanunquota/conf/master/centos7.sh
bash centos7.sh
```
Tested on
* CentOS 7 32 bit
* CentOS 7 64 bit
* OpenVZ only

### Service
* OpenSSH port 80
* Squid port 8080 (limit to IP VPS)
* DropBear port 443
* Anti Torrent & P0rn
